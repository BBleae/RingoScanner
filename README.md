# RingoScanner


## What is it?
RingoScanner is a framework for scanning the web for vulnerabilities.

## Requirements
- [Python 3.10.x](https://www.python.org/downloads/)
## Install
 ```bash
pip install -r requirements.txt
````

## Usage
```bash
python main.py
```
